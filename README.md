# Click-the-Button
This is a WinForms game "Click the Button", where you need to click a moving button as many times as possible within 30 seconds.

**Click the Button** is a small and fast-paced game made with **C# WinForms**.  
Your goal is simple: **click the button as many times as possible in 30 seconds**.

The button:
- randomly changes its position on every click  
- changes its color  
- always stays within the window bounds  

At the end of each round, the game shows your **best score** so far.
